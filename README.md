# SplitDivision---master-BillAPI

### Running the development server.

### To install

```bash
    npm install
```

### And run:

```bash
    npm run start
```

### To run project

#### Build docker image

```bash
    docker build -t master-bill-api .
```

#### Run a container on port 5003

```bash
    docker run -p 5003:5003 master-bill-api
```
